#promisify-github

Manipulating resources on Github by promises super easily.

## API

* delRepo

```javascript
/**
 * @param  {string} repo_name
 * @return {promise}
 */
function delRepo(repo_name)
```

* newRepo

```javascript
/**
 * newRepo
 * @param  {string} repo_name
 * @param  {string} desc
 * @return promise
 */
function newRepo(repo_name, desc)
```
